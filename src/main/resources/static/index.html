미확정(UNKNOWN) 결제 복구

웹훅 누락 등으로 방치된 결제를 PG 조회로 확정.

보상 태스크(망취소)

승인 후 재고 부족 시 자동 망취소. 소진(FAILED) 건 재시도.

정산 대사 불일치

사람 확인 필요(PENDING) 건 조회 + 수기 확정.

강제취소 · 2인 승인(maker-checker)

요청자와 승인자가 반드시 달라야 실행. 데모: 관리자 로그인 상태로 요청→목록→(다른 관리자) 승인.

FDS 사후 탐지 심사 큐

결제 완료 이벤트를 비동기 평가 → REVIEW/BLOCK을 큐에 적재. 거부 시 카드 블랙리스트.

DLQ (실패 이벤트)

멱등 컨슈머가 실패를 흡수한 데드레터. 재처리 가능.

정산 (수수료·부가세·지급예정일)

일 단위 집계 → 수수료 2.7% + 수수료 VAT 10% → 지급예정일(정산일+2영업일, 주말 skip). 집계된 정산(CREATED)을 지급 확정(PAID_OUT)한다. 구매확정(CONFIRMED)된 항목만 집계 대상.
